# MyButler
开发一个汇聚热门第三方的的App

最近做了了一个小项目，来复习一下Android基础
该项目叫《我的管家》，包括的功能如下：
使用热门的安卓技术，用Picasso、RXvolley、Bmob、Zxing等热门第三方SDK及框架，开发一个完整的包含有能语音陪聊，能看新闻，能查物流及电话号码归属地，能看美女相册，能扫描与生成个性化二维码，能定位与查看地图的生活类APP
具体用到的第三方如下：

 1.Bmob后端云
- 实现用户管理模块
2.智能聚合数据，科大讯飞TTS引擎
- 实现语音智能聊天
3.RXVolley网络框架
- 实现网络数据请求
4.Picasso图片库, PhotoView图片控件
- 实现美女图片加载缩放拖拽
5.Zxing二维码库
- 实现扫描和生成二维码
6.BaiduMap
- 实现地图定位，多图层绘制
7.自定义布局
- 实现时间轴效果
